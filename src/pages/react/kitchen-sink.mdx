# Kitchen Sink

Tailwind Mobile React Kitchen Sink is a great place to get started, it has a lot of examples for all Tailwind Mobile React components and covers most of aspects.

It is recommended to use it as a reference during development.

<Device src="/kitchen-sink/react/index.html" float />

## Source Code

React Kitchen Sink source code is located at [GitHub repository](https://github.com/tailwind-mobile/tailwind-mobile/tree/master/kitchen-sink/react)

## Running Locally

To run Tailwind Mobile React Kitchen Sink locally we need:

1. Clone (or download) [Tailwind Mobile GitHub repository](https://github.com/tailwind-mobile/tailwind-mobile)
2. Install required dependencies. Go to the folder with downloaded and Tailwind Mobile repository and execute in terminal:

   ```
   npm install
   ```

3. We are ready to launch Tailwind Mobile React Kitchen Sink, execute in terminal:
   ```
   npm run react
   ```
