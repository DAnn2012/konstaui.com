# Kitchen Sink

Tailwind Mobile [Vue Kitchen Sink](/kitchen-sink/vue/dist/index.html) is a great place to get started, it has a lot of examples for all Tailwind Mobile Vue components and covers most of aspects.

It is recommended to use it as a reference during development.

## Source Code

Vue Kitchen Sink source code is located at [GitHub repository](https://github.com/tailwind-mobile/tailwind-mobile/tree/master/kitchen-sink/vue)

## Running Locally

To run Tailwind Mobile Vue Kitchen Sink locally we need:

1. Clone (or download) [Tailwind Mobile GitHub repository](https://github.com/tailwind-mobile/tailwind-mobile)
2. Install required dependencies. Go to the folder with downloaded and Tailwind Mobile repository and execute in terminal:

   ```
   npm install
   ```

3. We are ready to launch Tailwind Mobile Vue Kitchen Sink, execute in terminal:
   ```
   npm run vue
   ```
